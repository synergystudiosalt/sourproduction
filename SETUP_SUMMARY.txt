QUICK START (5 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GET API KEYS (2 min)
   • Gemini: https://aistudio.google.com/app/apikey
   • Groq (optional): https://console.groq.com

2. PUSH TO GITHUB (1 min)
   $ git add .
   $ git commit -m "Add Cloudflare Pages Functions deployment"
   $ git push origin main

3. CREATE CLOUDFLARE PAGES PROJECT (2 min)
   • Go to https://dash.cloudflare.com
   • Pages → Create a project → Connect to Git
   • Root directory: sour.ai
   • Build command: npm run build
   • Build output directory: dist
   • Click "Save and Deploy"

4. ADD ENVIRONMENT VARIABLES (1 min)
   • Settings → Environment variables
   • Add GEMINI_API_KEY (or GEMINI_API_KEYS for multiple keys)
   • (Optional) Add GROQ_API_KEY
   • Save

5. REDEPLOY (1 min)
   • Deployments → Retry the failed build
   • Wait 2-3 minutes

6. VERIFY (1 min)
   • Visit https://your-project.pages.dev
   • Visit https://your-project.pages.dev/api/health
   • Try sending a chat message

HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Browser request to https://your-project.pages.dev
         ↓
Cloudflare Pages serves React app
         ↓
User sends chat/agent request
         ↓
Frontend calls fetch(apiUrl('/api/chat'))
         ↓
Cloudflare Pages Functions handler catches /api/* routes
         ↓
TypeScript handler in functions/api/chat.ts runs
         ↓
Calls Gemini API or Groq API (using your API keys)
         ↓
Returns response to frontend
         ↓
Frontend displays AI response
